# Botley Universal Discord Egg

<img src="logo.png" width="180">

A clean base egg for running **Python OR Node.js Discord bots**.

### Usage
Upload **one** bot file:
- `.py` → Python
- `.js` → Node.js

The egg automatically detects the correct runtime.
